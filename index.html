CSS Snippets Cheatsheet

Text Box in Image

This CSS Snippet allows for a box with a text inside of it

            .container {
            position: relative;
            }

            .text-block {
              position: absolute;
              bottom: 20px;
              right: 20px;
              background-color: black;
              color: white;
              padding-left: 20px;
              padding-right: 20px;
            }

Background and text color

This CSS Snippet is to create a simple box with a background color of red and a text color of white

              .box-red {      
              color: white;
              background-color: red;
              }

Loader Icon

This CSS Snippet is used to create a "loader"

              .loader {
              border-top: 16px solid blue;
              border-right: 16px solid green;
              border-bottom: 16px solid red;
              }

Image Magnifier Glass

Creates a magnifying glass when viewer hovers over the image.

            * {box-sizing: border-box;}

            .img-magnifier-container {
            position: relative;}

            .img-magnifier-glass {
            position: absolute;
            border: 3px solid #000;
            border-radius: 50%;
            cursor: none;
            /*Set the size of the magnifier glass:*/
            width: 100px;
            height: 100px;
            }

Image Rounding

Creates a round image.

            img {
            border-radius: 50%;
            }

Side by Side Imaging

Sets two images side by side.

            .column {
              float: left;
              width: 33.33%;
              padding: 5px;
            }
            
            .row::after {
              content: "";
              clear: both;
              display: table;
            }
